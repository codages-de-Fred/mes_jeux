Les règles du jeu

but du jeu

Le but du jeu est d'avancer le trait sans toucher le trait adverse ni toucher son propre trait ni toucher les bords du jeu.
On ne peut donc pas faire marche arrière.

Le jeu se joue à 2 joueurs uniquement.

Les touches directionnelles

Le trait bleu

vers le haut
touche clavier "Z"

vers la gauche
touche clavier "Q"

vers le droite
touche clavier "D"

vers le bas
touche clavier "X"

Le trait rouge

vers le haut
touche clavier numérique "8"

vers la gauche
touche clavier numérique "4"

vers le droite
touche clavier numérique "6"

vers le bas
touche clavier numérique "2"
